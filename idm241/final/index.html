Happy Cog Microinteraction
Enhancement: Final Description

Triggers

  • hover over / off menu bars
  • hover over / off menu items
  • click on / off menu items
  • hover over / off menu bars with selected menu item
  • 5th rule would is not beneficial to my menu, approved by Jervo

Rules

On State

  • Hover Menu Bars
  • cursor changes to finger pointer
  • bottom menu bar shifts down far enough to fit all menu items
  • menu items fade in & down in a decsending order beginning with the first menu item
  • Hover Menu Items
  • Menu item text color eases-in from white #FFFFFF to their brand yellow #F1CF67 while:
  • Menu item glides from being right aligned to being left aligned
  • Click Menu Items
  • Asterisk instantly appears on the left of the selected menu item
  • Hover Menu Bars Selected Item
  • selected menu item between two menu bars quickly fades out
  • cursor changes to finger pointer
  • bottom menu bar shifts down far enough to fit all menu items
  • menu items fade in & down in a decsending order beginning with the first menu item

Off State

  • Hover Menu Bars
  • cursor changes back to arrow cursor
  • bottom menu bar shifts up to its original position while:
  • menu items fade out & up in an acsending order beginning with the last menu item
  • Hover Menu Items
  • Menu item text color eases-in from their brand yellow #F1CF67 to white #FFFFFF while:
  • Menu item glides from being left aligned to being right aligned
  • Click Menu Items
  • Asterisk instantly disappears from the left of the selected menu item
  • Hover Menu Bars Selected Item
  • cursor changes back to arrow cursor
  • bottom menu bar shifts up to its original position while:
  • menu items fade out & up in an acsending order beginning with the last menu item
  • selected menu item quickly fades in between the two menu bars

Loops & Modes

  • Hover On / Off Menu Bars
  • menu items appear on hover / disappear off hover
  • Hover On / Off Menu Bars Selected Item
  • menu items appear on hover / disappear off hover
